JS API Events

Enable JS API Events

Dispatch custom events to page window

Available Events:
scrapfly:ready Extension loaded and initialized
scrapfly:onStart Detection started (cache miss / manual)
scrapfly:onProgress Detection progress updates (method-level)
scrapfly:onHooksComplete JS hooks monitoring completed
scrapfly:onWindowPropsComplete Window property monitoring completed
scrapfly:onDetection Detection completed with results
scrapfly:onError Detection error occurred
Usage Example: 
window.addEventListener('scrapfly:onStart', (e) => {
  console.log('Detection started:', e.detail.url);
});

window.addEventListener('scrapfly:onDetection', (e) => {
  console.log('Detections:', e.detail.detections);
  console.log('URL:', e.detail.url);
});

Duplicate Prevention

Enable Duplicate Prevention

Block duplicate history entries within a time window

Scope
Time Window
How it works:

Prevents duplicate history entries for the same URL within the time window.

  • Full URL: Exact URL match (including query params)
  • Domain: Same domain (different paths counted as same)
  • Path: Same domain+path (ignores query params)
